# 常用 Linux 文件与目录操作命令

| 命令                          | 作用说明                           |
|------------------------------|------------------------------------|
| `cd`                         | 进入指定目录                       |
| `pwd`                        | 显示当前所在目录                   |
| `ls`                         | 查看目录内容                       |
| `touch file1 file2 file3`    | 新建多个文件（文件名之间用空格分隔） |
| `cat file1`                  | 查看文件内容                       |
| `rm file3`                   | 删除文件                           |
| `mkdir dir1 dir2 dir3`       | 新建多个目录                       |
| `cp file2 dir3`              | 复制文件到指定目录                 |
| `mv file2 dir3`              | 移动文件到指定目录                 |
| `mv file1 dir1/file11`       | 移动文件并重命名                   |
| `mkdir dir4`                 | 新建一个空目录                     |
| `rmdir dir4`                 | 删除一个空目录                     |
| `rm -r dir1`                 | 删除一个非空目录                   |
| `man + 命令名`            | 查看命令的详细说明                 |
